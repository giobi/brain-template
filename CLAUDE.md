# Claude Code Configuration

**Leggi e segui le istruzioni in `AGENTS.md`** - contiene il sistema completo per organizzare questo brain.

Questo file esiste per compatibilità con Claude Code. Il cuore del sistema è in AGENTS.md, che funziona con qualsiasi agent AI (Claude Code, Codex CLI, ecc.).

## Quick Start

Se è la prima sessione:
1. Leggi `AGENTS.md`
2. Fai le domande di configurazione all'utente
3. Crea la struttura base del brain
4. Mostra riepilogo

## Riferimenti

- **Setup e struttura**: `AGENTS.md`
- **Identità agent**: `boot/identity.md` (dopo configurazione)
- **Database**: `database/people/`, `database/projects/`
- **Contenuti**: `diary/`, `log/`, `todo/`
