# Persone

Indice di tutte le persone nel database.

## Schema Consigliato

Ogni persona deve avere:
- `name`: Nome completo
- `tags`: Relazione (amico, collega, cliente, famiglia)
- `contact`: Email, telefono, social
- `notes`: Note libere
